「GPU 每小時 $0.29 起」——看到這種廣告先別急著開機 💸

最低時薪 ≠ 最低成本。真正該算的公式是：

真實成本 = GPU 運算 + 儲存 + 傳輸 + 環境設定 + 等待 + 失敗重跑

三個常見情境：

🔹 4 小時的 ComfyUI 互動工作：時薪差 $0.2，總共差不到 $1 —— 但如果換到便宜主機要重建環境、重載模型一小時，省的錢直接賠掉

🔹 停機後保留 100GB 模型權重：運算歸零，儲存不歸零。有些平台閒置磁碟收 $0.10–0.20/GB/月，一個月默默多 $10–20 美金

🔹 72 小時的 H100 訓練：這時候時薪才真的重要（差 $1/hr = 差 $72），但前提是任務有 checkpoint，失敗才不會從頭來過

租便宜 GPU 前先問三個問題：
1️⃣ 這到底是哪台機器？
2️⃣ 停機之後還要付什麼錢？
3️⃣ 任務中斷後能不能接著跑？
