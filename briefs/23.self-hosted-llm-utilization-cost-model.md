**Status:** Backlog. Requires transparent assumptions and current pricing.

**ICP:** Product, finance, and infrastructure owners deciding between APIs and self-hosted inference.

**Search intent:** Consideration and decision - self-hosted LLM versus API cost and break-even.

**Working title:** When Does Self-Hosting an LLM Beat API Pricing? Model the Cost of Idle GPUs Too

**Required evidence:** input/output-token assumptions; workload request rate; model and quality trade-off; API price/date; GPU rate/date; measured or conservative throughput; utilization and redundancy scenarios; formula or calculator.

**Outline:** answer-first decision rule; why utilization changes the answer; scenario table at 1/10/100 RPS; break-even calculator; non-cost trade-offs; Glows.ai deployment option; caveats.
