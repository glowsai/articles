**MimicTalk** 是文字轉語音（TTS）工具，可生成自然、接近真人的語音。它支援多種語言、語調與自訂選項，可用於虛擬助理、內容製作和教育應用。

使用 **Glows.ai**，不需自行處理繁複的安裝與設定，即可在雲端環境執行 MimicTalk。本指南說明如何設定 MimicTalk，並透過 HTTP 或 SSH 連線。

## **步驟一：登入 Glows.ai 並啟動 MimicTalk 環境**

1. **註冊並登入**：前往 [Glows.ai](https://glows.ai/)，註冊帳號並登入。
2. **啟動 MimicTalk 環境**：在控制台找到 **MimicTalk** 工作範本，點選「Complete Checkout」。Glows.ai 會分配資源並設定相依套件。

## **步驟二：存取 MimicTalk 環境**

環境啟動後，可透過以下方式存取：

### **1. HTTP WebUI（Port 7860）**

這是用來生成語音的 MimicTalk 主要網頁介面。

- 開啟 URL：
- plaintext
- [`https://tw-03.access.glows.ai:24527?token=iOuz4vDMME`](https://tw-03.access.glows.ai:24527/?token=iOuz4vDMME)
- 你可以在介面中輸入文字並調整語音參數。

### **2. SSH 存取（Port 22）**

SSH 提供後端環境的命令列存取方式，適合進階操作。

- SSH Command：
- `ssh user@<your-instance> -p 22`
- **密碼**：請查看 Glows.ai 控制台中的執行個體詳細資料。

## **步驟四：儲存並匯出輸出內容**

1. **從 WebUI 下載檔案**：使用 MimicTalk WebUI 的 **Download** 按鈕，將生成的音訊檔案儲存至本機裝置。
2. **透過 SSH 管理檔案**：使用 SCP（Secure Copy）將伺服器檔案傳輸到本機：`*scp -P 22 user@<your-instance>:~/output.wav ./output.wav*`
3. **輸出路徑：** `infer_out/tmp/demo.mp4`
