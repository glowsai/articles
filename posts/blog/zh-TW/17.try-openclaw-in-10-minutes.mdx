## OpenClaw 是什麼？為什麼大家都在討論它？

OpenClaw 是一套開源、自架的 AI Agent 閘道：把你日常使用的通訊軟體——Telegram、WhatsApp、Discord、Slack 等——接上一個能在你掌控的機器上執行實際工具的 AI 模型。跟只能「回話」的聊天機器人不同，OpenClaw 的 Agent 可以管理檔案、執行指令、瀏覽網頁、跑排程自動化。模型負責決策，OpenClaw 負責執行。

這波熱度是有數字可查的。專案在 2026 年 1 月 25 日以「Clawdbot」為名發布，24 小時內拿下 9,000 顆 GitHub 星星（[Elephas](https://elephas.app/blog/opean-claw-clawdbot-viral-launch)，2026 年 1 月）。一週內被迫改名兩次——Clawdbot → Moltbot → OpenClaw——熱度反而加速，約三週內從 2,000 星飆到超過 168,000 星（[HackerNoon](https://hackernoon.com/why-everyone-is-panic-buying-mac-minis-for-openclaw-moltbot-clawdbot)，2026 年 2 月）。作者 Peter Steinberger 曾出售 PDF 工具公司 PSPDFKit，第一版只花了大約一小時就寫出來（[36kr](https://eu.36kr.com/en/p/3655938014093701)，2026）。周邊效應也上了新聞：Solana 上的假代幣市值一度衝到 1,600 萬美元，Slashdot 則在 2026 年 1 月 28 日刊出「[『Clawdbot』讓 AI 圈搶購 Mac Mini](https://tech.slashdot.org/story/26/01/28/0510226/clawdbot-has-ai-techies-buying-mac-minis)」。

最後那一條正是本文要解決的問題：想知道大家在瘋什麼，真的不需要先刷一台 Mac Mini。

## 為什麼 OpenClaw 需要真正的硬體（和一顆 AI 大腦）

OpenClaw 本體很輕量：閘道只需要 Node.js 22+ 和 4 GB RAM 就能運作（[Cherry Servers](https://www.cherryservers.com/blog/openclaw-hardware-requirements)，2026），連 Raspberry Pi 理論上都跑得動。那為什麼大家搶著買要價 US$599 的桌機？

因為閘道不等於 Agent。OpenClaw 背後需要一個模型，而且需要一台你放心給它 shell 權限、可以長時間開機的機器。「大腦」只有兩種來源：

**選項一：雲端 API 金鑰。** 簡單，但按量計費。Anthropic 自 2026 年 4 月 4 日起封鎖 Claude Pro 與 Max 訂閱在 OpenClaw 等第三方 Agent 框架的使用，把用戶推向按量 API 費率——Claude Sonnet 4.6 每百萬輸入 token US$3、每百萬輸出 US$15（[The Next Web](https://thenextweb.com/news/anthropic-openclaw-claude-subscription-ban-cost)，2026）。同一篇報導引述估算：一個全天自主運作的 OpenClaw 實例，依任務量不同可能消耗相當於 US$1,000–5,000 的 API 額度。一般玩家的實際帳單遠低於此，但重點不變：讓一個全天候 Agent 跑在按量計費的 token 上，是一筆上不封頂的開銷。

**選項二：本地模型。** 每則訊息零成本，提示詞也不會離開你的機器——但前提是硬體裝得下模型權重。這正是 Mac Mini 搶購潮的真正原因：統一記憶體塞得下一個堪用的本地模型，又安靜省電。而另一條熱門路線——每月 US$5–20 的純 CPU VPS（[Analytics Vidhya](https://www.analyticsvidhya.com/blog/2026/02/openclaw-hosting-mac-mini-vs-cloud-vps/)，2026 年 2 月）——裝得下閘道、裝不下像樣的模型，最後還是把你推回選項一和它的 token 帳單。

租用 GPU 實例是第三條路：享有本地模型的成本結構，卻不必把硬體買回家。

## 10 分鐘上手路徑：Glows.ai 官方 OpenClaw 映像檔

Glows.ai 維護一個預裝好的 OpenClaw 映像檔——OpenClaw、Ollama、JupyterLab 全部就位——試用路徑因此跳過所有安裝步驟。完整圖文版請見教學文章〈How to Run OpenClaw on Glows.ai〉，以下是濃縮版：

1. **註冊並建立實例。** 到 [Glows.ai](https://glows.ai) 點選 `Create New`，Workload Type 選 Inference GPU — 4090，映像檔選官方 **OpenClaw**（[建立實例教學](https://docs.glows.ai/docs/create-new)）。點 `Complete Checkout` 後，開機需時 30–60 秒。
2. **打開終端機。** 在 `My Instances` 開啟 `HTTP Port 8888` 的連結——那是 JupyterLab——並建立一個 Terminal。Port 22 是 SSH，Port 11434 是 Ollama API。
3. **執行初始設定。** `openclaw onboard` 會引導你同意條款，並可選擇性設定 Telegram 機器人 Token（向 `@BotFather` 申請）。模型設定先留空，下一步會處理。
4. **啟動本地模型。** 一行指令就能下載模型並啟動 OpenClaw 服務：

```bash
ollama launch openclaw --model qwen3.5:4b
```

5. **開始對話。** 三個入口立即可用：直接在 CLI 對話、完成一行驗證指令後在 Telegram 機器人對話，或透過 SSH 連接埠轉發用瀏覽器開啟 Web UI。

**Note:** 4B 模型是下載快的示範選擇。你租 24 GB RTX 4090 而不是 CPU 主機的理由正是餘裕——同一個實例可以換上大得多的本地模型，讓 Agent 推理能力更好；哪張卡裝得下哪種規模的模型，可參考我們的文章〈本地 LLM 主機 vs 雲端 GPU 成本：2026 自組電腦實價試算〉。

**Reminder:** 計費以小時為單位。實驗結束記得關閉實例，計費就停止。

## 常見問題

### OpenClaw 是什麼？

OpenClaw 是一套開源、自架的 AI Agent 閘道，把 Telegram、WhatsApp、Discord 等通訊軟體接上一個擁有實際工具權限的 AI 模型——可以管理檔案、執行指令、瀏覽網頁、跑排程自動化。它在 2026 年 1 月以 Clawdbot 之名發布，數週內 GitHub 星星數突破 168,000。

### 一定要買 Mac Mini 才能跑 OpenClaw 嗎？

不用。Mac Mini 之所以流行，是因為統一記憶體裝得下本地模型、又能安靜地 24 小時運作；但任何具備 Node.js 22+ 和 4 GB RAM 的機器都能跑閘道——而租一台雲端 GPU 就能獲得本地模型能力，不必花 US$599（約 NT$1.9 萬）買硬體。

### 試用 OpenClaw 要花多少錢？

兩小時大約 US$1（約 NT$31）。Glows.ai 的 OpenClaw 預裝映像檔在 RTX 4090 上每小時 US$0.49 起（2026 年 7 月查核），內建 Ollama，所以沒有 API token 費用，也沒有月費方案。

### OpenClaw 可以用本地模型取代 Claude API 金鑰嗎？

可以。在 Glows.ai 的 OpenClaw 映像檔上，一行指令——`ollama launch openclaw --model qwen3.5:4b`——就能下載本地模型並啟動服務。每則訊息零成本，提示詞也留在你的實例上；考量到 Anthropic 已於 2026 年 4 月切斷第三方 Agent 的 Claude 訂閱存取，這點格外重要。
