# RunPodの代替案：時給より先に確認する9項目

RunPodの代替は、何を代替したいかで変わります。Serverless、テンプレート、安価なバッチGPU、または対話型ワークスペースでは候補が異なります。

1. 必要なVRAMはいくつか
2. 同じGPU・同じリージョンを比較しているか
3. 課金はいつ開始・停止するか
4. 停止後にデータは残るか
5. 必要な時にGPUを確保できるか
6. ジョブは中断・再開できるか
7. ユーザーとデータはどこにあるか
8. Serverlessか永続マシンか
9. 環境、監視、障害を誰が担当するか

| ニーズ | 先に評価する方向 |
| --- | --- |
| checkpoint可能な低コスト処理 | Vast.aiなどのマーケットプレイス |
| 大規模・複数GPU学習 | Lambda、Nebiusなど |
| ノートブック中心の開発 | Paperspace系 |
| 台湾の対話型GPU作業 | Glows.ai |
| トークン課金の推論 | DeepInfraなどのAPI |

各候補で同じ60分タスクを実行し、起動、最初の出力、保存、停止、請求を記録してください。詳しくは[RunPod vs Glows.ai](https://glows.ai/article/runpod_vs_glows_ai_ja)も参照できます。
